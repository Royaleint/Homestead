# Homestead

See housing decor vendors and their wares on your map and track your collection progress.

## Features

- Map pins for 238+ housing decor vendors
- Mouse over pins to see vendor inventory
- Track owned vs missing decor items
- Minimap button for quick access to options
- Export/import scanned vendor data
- Validation tools for data integrity

## Installation

1. Download and extract to your AddOns folder
2. Enable Homestead in your addon list
3. Open your map to see vendor pins

## Commands

| Command | Description |
|---------|-------------|
| `/hs` | Show help |
| `/hs scan` | Scan targeted vendor |
| `/hs validate` | Check data integrity |
| `/hs export` | Export scanned data |
| `/hs import` | Import community data |

## Known Limitations (Alpha)

- Ownership tracking requires opening the Housing Catalog UI once per session (Blizzard API limitation)
- Some vendors may have placeholder coordinates

## Contributing

Scanned vendor data welcome! Use `/hs export` and submit via GitHub issues.
